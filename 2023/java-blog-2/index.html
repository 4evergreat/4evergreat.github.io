用javac和java命令输出HelloWord

11月24日, 2023年
阅读时长: 1 分钟 # 共348 字
JavaSE

代码（用控制台便运行文件） 

1
2
3
4
5

public class Hello{
	public static void main(String[] args){
		System.out.print("Hello Word!");
	}
}

1. 创建文件

  • 文件后缀名为.java

  • Hello.java

  • [注意点]系统可能没有显示文件后缀名，我们需要手动打开

    窗口状态栏=>查看=>显示=>文件扩展名（勾选）

2. 编译 : javac [文件名.java] 

1
2

C:\Users\Administrator\Desktop\新建文件夹\java>javac Hello.java
//会生成 Hello.class 文件

3. 运行： java [文件名] 

1
2

C:\Users\Administrator\Desktop\新建文件夹\java>java Hello
Hello Word!

在运行 Hello.class文件，但是不需要添加文件后缀

可能会遇到的情况

  1. Java大小写敏感， 如上述代码中 System, String[], public 的大小写区分
  2. 尽量使用英文
  3. 文件名 和 类名必须保证一致，并且首字母大写
  4. 符号使用的了中文
